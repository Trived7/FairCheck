# FairCheck — AI Bias Detection Tool

FairCheck is a browser-based tool that helps organizations 
detect hidden bias in datasets before they are used in 
automated decision-making systems like hiring, loans, or 
medical care.

Upload any CSV file and get instant bias analysis across 
sensitive attributes — gender, race, age, income, and more.

## Live Demo
https://astounding-tiramisu-e979d5.netlify.app

## Features
- CSV upload with drag and drop
- Automatic sensitive column detection
- Bias score per column (0-100) with risk classification
- Visual charts for bias comparison
- Suggested fixes for each flagged column
- Gemini AI deep analysis and recommendations
- Built-in sample datasets for hiring, loans, and medical records

## Built With
- HTML, CSS, JavaScript
- Chart.js for visualizations
- Gemini 2.0 Flash API
